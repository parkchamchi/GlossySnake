# Backend

## venv

### Create
```
python -m venv .venv
```

### Activate/Deactivate
Use `venv.bat` and `venv_d.bat` in this directory.

### Install dependencies
```
pip install -r requirements.txt
```

Installed: `django`, `djangorestframework`, `django-cors-headers`

### Make (or reset) a DB
Use `dbreset.db`.

### Run
Use `runserver.bat`.
